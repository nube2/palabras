# Palabras

A single-page Spanish vocabulary practice site. Choose an A2, B1, or B2 level
and two, three, or four words to receive a random set taken from a sentence at
that level. Sentences are grouped by level in `data/sentences.js`. Every
selected word is distinct and at least four letters long. Use the Spanish help
button to reveal the source sentence.

No build step or dependencies are required. Open `index.html` or serve the
directory with any static file server.
