# Palabras

A single-page Spanish vocabulary practice site. Choose two, three, or four
words and receive a random set from a local database of the 2,000
highest-frequency Spanish word forms.

The frequency data is derived from the Spanish list in
[FrequencyWords](https://github.com/hermitdave/FrequencyWords), which is based
on the OpenSubtitles corpus.

No build step or dependencies are required. Open `index.html` or serve the
directory with any static file server.
